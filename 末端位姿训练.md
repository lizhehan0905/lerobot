从机械臂获取到的机械臂信息 GetArmEndPoseMsgs 单位
'''获取机械臂末端位姿消息

X,Y,Z单位0.001mm
RX,RY,RZ单位0.001度
'''
'''Retrieves the end effector pose message of the roboticarm.
This includes the following information:
    X, Y, Z position (in 0.001 mm)
    RX, RY, RZ orientation (in 0.001 degrees)
'''

说明机械臂得到的数据是0.001度，举例: RX=90000，实际角度：90度

当时为了得到小数的末端位姿，将SDK中的数据集都进行了 *0.000001 操作
所以数据集中的数据单位是 1000度，例如 0.09 实际角度是 90度

```python
end_pos_state = self.piper.GetArmEndPoseMsgs() # 末端位姿
self.motors_eef["end_x.epos"] = round(end_pos_state.end_pose.X_axis * 0.000001,8)     #单位为1000mm
self.motors_eef["end_y.epos"] = round(end_pos_state.end_pose.Y_axis * 0.000001,8)
self.motors_eef["end_z.epos"] = round(end_pos_state.end_pose.Z_axis * 0.000001,8)
self.motors_eef["end_roll.epos"] = round(end_pos_state.end_pose.RX_axis * 0.000001,8) #单位为1000度
self.motors_eef["end_pitch.epos"] = round(end_pos_state.end_pose.RY_axis * 0.000001,8)
self.motors_eef["end_yaw.epos"] = round(end_pos_state.end_pose.RZ_axis * 0.000001,8)
self.motors_eef["end_gripper.epos"] = self.motors["gripper.pos"]
```
注意，上面所述度都是角度

### 欧拉角转Rotate6D

```python
from scipy.spatial.transform import Rotation as R

def euler_to_rotate6d(q: np.ndarray, pattern: str = "xyz") -> np.ndarray:
    return R.from_euler(pattern, q, degrees=False).as_matrix()[..., :, :2].reshape(q.shape[:-1] + (6,))
```

### Rotate6D 转 xyz
```python
def rotate6d_to_xyz(v6: np.ndarray) -> np.ndarray:
    v6 = np.asarray(v6)
    if v6.shape[-1] != 6:
        raise ValueError("Last dimension must be 6 (got %s)" % (v6.shape[-1],))
    a1 = v6[..., 0:5:2]
    a2 = v6[..., 1:6:2]
    b1 = a1 / np.linalg.norm(a1, axis=-1, keepdims=True)
    proj = np.sum(b1 * a2, axis=-1, keepdims=True) * b1
    b2 = a2 - proj
    b2 = b2 / np.linalg.norm(b2, axis=-1, keepdims=True)
    b3 = np.cross(b1, b2)
    rot_mats = np.stack((b1, b2, b3), axis=-1)      # shape (..., 3, 3)
    return R.from_matrix(rot_mats).as_euler('xyz',degrees=True)
```
